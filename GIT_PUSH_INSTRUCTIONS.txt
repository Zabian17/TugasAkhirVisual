================================================================================
TROUBLESHOOTING
================================================================================

Q: Error "branch already exists"
A: Use: git checkout "update 2.0"
   (instead of -b to switch to existing branch)

Q: Error "nothing to commit"
A: Check: git status
   Ensure files are staged: git add -A

Q: PROGRESS.txt is in staging area
A: Remove it: git reset PROGRESS.txt
   It should not be staged (it's in .gitignore)

Q: Push fails with permission error
A: Check your GitHub credentials
   Or use: git push -u origin "update 2.0" --force (NOT RECOMMENDED)

Q: Can't create branch
A: Make sure you're in correct directory
   And git is initialized: git rev-parse --git-dir
